# State-of-Being: Negative sentences

### Negative sentences

```text
// <Subject Pronoun> + は + <Noun> + では + ありません

私は学生ではありません。

わたしはがくせいではありません。

watashi wa gakusei dewa arimasen.

I'm not a student.
```

* `じゃありません` is also used instead of `ではありません`. `じゃありません` sounds more casual.
  * わたしががくせいじゃありません
